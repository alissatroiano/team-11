Image Map as Background Image While Retaining Clickable Areas

Client Side Image Map

This image map was made using the rectangular tool where only the name of the state is used to produce the clickable area. Anywhere off these areas will produce a clickable link back to home page. The original image map is being used as a background image with a clear image map over it which retains the clickable areas. You must have a good graphics editor which allows you to set a copy of your image map to full transparency. Be sure to save as a PNG/GIF image. May have to tweak to re-align the clickable areas..

Clicking on the state names will open new tab to Wikipedia for that state.
